# LANCOF Project Page

Project page for:

**Length-Agnostic Open-Vocab Video Amodal Completion via Diffusion Models with Text and Optical Flow Guidance (LANCOF)**.

## Publish on GitHub Pages
1. Push this folder as a GitHub repo (recommended) or copy its contents to your repo root.
2. In GitHub: `Settings → Pages → Build and deployment`, select your branch and set **Folder** to `/ (root)`.
3. Update the paper/code links in `index.html`.

Notes:
- `.nojekyll` is included for GitHub Pages.
- Replace `static/paper.pdf` with your actual PDF when ready.
